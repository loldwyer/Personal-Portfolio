Project Overview (Assignment 1)

This portfolio website was developed as part of the EEN1037_20732 Web App Development course. It serves as a professional platform to showcase my projects, experience, blog, resume, and contact details while offering an interactive and user-friendly experience.

Technologies Used

  • HTML5 – Structure and content.
  • CSS3 – Custom styling and layout.
  • Local Storage – Previously used for testimonials (now replaced by a blog).
  • Responsive Design – Optimized for desktop & mobile.
  • Static Pages Only – No JavaScript for core functionality.

Project Features

  • Interactive Navigation: Dropdown menus and profile-based quick contact.
  • Resume Page: Displays my qualifications and work experience.
  • Blog: Displays multiple blog posts, with the latest previewed on the homepage.
  • Projects Page: Lists all my projects, each linking to a detailed project description.
  • Project Details: Each project has its own page with images, descriptions, and clickable images for easy viewing.
  • Contact & Feedback: Dedicated contact page and a simple feedback form.

Assignment 2 Overview

In Assignment 2, the Personal Portfolio Website has been extended to include client-side interactivity and an Admin Dashboard, enabling real-time content management.

  • Secure Admin Dashboard for managing projects, blog posts, and the resume.
  • Real-time UI updates with JavaScript-based event handling.
  • Enhanced resume editor and blog posting with font customization and PDF export.
  • Blog post interaction system (likes and comments).
